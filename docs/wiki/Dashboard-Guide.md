# Dashboard Guide

The Terminal User Interface (TUI) provides real-time visibility into the engine.

## Views
- **Status Panel**: Displays current state, round, and mode.
- **Log Tail**: Shows live event stream.
- **Metrics**: Displays generation success rates, VRAM usage, and training loss.

## Key Bindings
- `q`: Quit dashboard.
- `f`: Trigger "train-now" action.
- `r`: Toggle between raw and clean data views.
